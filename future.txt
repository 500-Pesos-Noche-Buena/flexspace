LANDING PAGE OPTIMIZATION (CRITICAL FOR 5000+ SPACES)
================================================================================

PROBLEM: Loading 5000+ spaces will cause LAG

SOLUTION: Fetch spaces based on USER'S LOCATION

APPROACH:
1. Get user's location via browser geolocation
2. Find nearby districts/cities
3. Load ONLY spaces in that area (20-50 spaces max)
4. User can change city/district manually

RESULT:
- Faster page load
- Relevant results
- Scalable to 10,000+ spaces
